# Kernel Debugging Tools Directory

This directory contains advanced kernel-mode debugging tools and utilities.

## Structure

- `ioctl/` - IOCTL detection and analysis tools
- `firmware/` - Firmware extraction and analysis utilities  
- `breakpoints/` - Advanced breakpoint and tracing systems
- `fuzzing/` - Kernel fuzzing infrastructure
- `plugins/` - Debugger and analysis tool plugins
- `vmkit/` - VM orchestration and parallel execution
- `scripts/` - Helper scripts and utilities

## Integration

These tools integrate with the existing pf task runner system and binary lifting infrastructure.
